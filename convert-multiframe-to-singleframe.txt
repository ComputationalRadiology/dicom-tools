
A good tool for converting from MultiFrame DICOM to single frame is emf2sf
from the dcm4che-tools package.

docker pull dcm4che/dcm4che-tools

# To convert multiframe to a series of dicom files:
docker run --rm -it --volume `pwd`:/data dcm4che/dcm4che-tools emf2sf \
 --out-file /data/singleframe-000.dcm /data/multiframe.dcm


# To transmit to research PACS:
docker run --rm -it -v `pwd`:/data crl/dicom-tools dcmsend --verbose 10.20.2.22 104 -aet COMPRESEARCH01 mri-*.dcm

# To transmit to local host storage SCP :
docker run --network=host --rm -it -v `pwd`:/data crl/dicom-tools dcmsend  --verbose 127.0.0.1 11112 -aet COMPRES *.dcm
